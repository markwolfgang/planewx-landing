# Screenshots Directory

Place your screenshot images here.

## Recommended:
- **Format**: PNG or JPG
- **Size**: 1200-1600px wide (for best quality on retina displays)
- **Naming**: 
  - `screenshot-1.png`
  - `screenshot-2.png`
  - `screenshot-3.png`
  - etc.

## Usage:
The landing page will automatically display screenshots from this directory. Update the `src` attributes in `components/landing-page.tsx` to match your actual screenshot filenames.

## Tips:
- Use high-quality screenshots that showcase key features
- Consider adding subtle borders or shadows in the images
- Keep file sizes reasonable (< 500KB per image) for fast loading
- Use descriptive alt text for accessibility
